SUNO AI API
I’ve created my own Suno API, and you can do the same! Suno is one of the best tools for music generation, but the challenge is that it doesn’t officially offer an API yet. However, I found a great workaround that lets you build an API using your personal Suno account. You can generate 10 tracks daily for free; it’s cool! Issues with music generation + hot tip
I've been working with the Suno API through PiAPI but the last few days the service has been super unreliable and basically down. TLDR; Suno doesn't have direct API access so you have to go through one of these 3rd party services. My best guess is that they are just getting too overlogged and are giving low priority to these services because using the web interface works fine. I'm curious if anyone has tried a different service like sunoapi.org or the roll-your-own version gciu.ai. I've had some suggestions in another forum to talk to PiAPI support so I may try that. In the meantime, I found an alternative music generator that is currently in beta and free. It's way faster than Suno and pretty close in quality. Also it didn't spit back my disco version of Jabberwocky like Suno did. https://www.riffusion.com/song/35232c08-4819-415e-b497-d9b3751cafaf
