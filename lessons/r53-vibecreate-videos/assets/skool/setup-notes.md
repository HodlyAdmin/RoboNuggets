How to install Claude Code + Remotion

step 1: install Claude Code

Get VSCode and install it - https://code.visualstudio.com/download

Find the 'Claude Code' extension and install it

step 2: install Remotion

to 'install' this skill, you need to run npx skills add remotion-dev/skills in the terminal of VSCode

To run that though, you need 2 prerequisites to be installed in your device:

node JS (lets you run the "npx" shortcut): https://nodejs.org/en/download

install GIT (lets you access "remotion-dev/skills" over at github.com): https://git-scm.com/install/

Custom GPT link: https://chatgpt.com/g/g-6972ab1405548191be9a2fada4e0770b-claude-code-x-remotion-prompter-by-robonuggets
