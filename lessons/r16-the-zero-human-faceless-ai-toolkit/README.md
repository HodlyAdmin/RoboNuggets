Prompts & References

prompt for ideas:

My video niche is [Batman in different superhero costumes]

To describe it in more detail: 

[I want different Batman costumes based on countries and their flags. Describe the background in a lot of detail and also make Batman do epic poses. The prompts should be hyper-realistic and very cinematic]

--

Give me 5 video prompts in this table format 




these are the columns: 

id, status, title, caption, prompt




id goes from 1 onward

status is always "todo"

title is a short phrase to describe the video

caption is optimised for tiktok, instagram, and youtube

prompt is a detailed description which you can provide to an ai video generation tool 




Never use double quotes for any of the values

reference on ayrshare json structure

{
  "post": "(insert caption here)",
  "mediaUrls": [
    "(map webhook url here)"
  ],
  "platforms": [
    "tiktok"
  ]
}
