Tools

Antigravity: https://antigravity.google/

Clipping tool options:

Vizard.ai — AI-powered clipping with custom templates, subtitles, and a strong API for automation. https://vizard.ai

Opus Clip — Auto-imports from YouTube with strong AI clip detection and Zapier integration. https://opus.pro

Submagic — API-first clipping tool with great caption styling. https://submagic.co

FFmpeg — Free, open-source video tool that gives you total creative control, but you build everything yourself.

This lesson uses Vizard.ai as the clipping engine. The agent template is pre-configured for Vizard, but the architecture can be adapted for other tools. 

Airtable — Spreadsheet-style database used as your clip review queue, where you approve or reject shorts from your phone or desktop. https://www.airtable.com/  | How to set up AirTable Credential

Telegram — Free messaging app where you create a bot to trigger and interact with your clipping pipeline. https://telegram.me/BotFather

Modal — Serverless cloud platform that runs your bot 24/7 — it only spins up when there's a video to process, so it costs almost nothing. https://modal.com/

Blotato — Multi-platform social media scheduler that auto-posts your approved clips to YouTube Shorts, TikTok, Reels, and more. https://blotato.com/
