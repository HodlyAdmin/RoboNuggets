# R58 | Antigravity + Figma Mobile App Builder

This module keeps the full lesson starter kit local and runnable as a scaffold. It preserves the example apps, MCP guide, and setup prompt while keeping Figma itself as an optional provider for the next implementation phase.

## What works now

- `npm run r58:dry` writes the local plan, manifest, and report.
- `npm run r58` seeds a local workspace with the downloaded starter files and reference prompt.

## Current blockers

- Live Figma MCP usage still needs authentication.
- The actual design-to-code runtime needs a later implementation pass once the target stack is chosen.
